Step 1 — Prepare your file

Download the TailWag template, or export from your existing system and follow the guide below. Any CSV with owner and dog columns will work — column names don't have to match exactly.

⬇ Download Template (.xlsx) Open in Excel, Numbers, or Google Sheets → fill in the Import Template tab → save as CSV → upload here

Exporting from your current system

🐾 Gingr
Reports → Client List
Include: Owner name, phone, email, pet name, breed, age
Export as CSV → rename columns to match template
📋 PetExec
Reports → Client/Pet Report
Select all columns → Export
Rename headers as needed
📑 Spreadsheet / Paper
Open the template
Type in each pet parent + dog
One row per dog (repeat owner info)

💡 Column names are flexible — TailWag recognizes "First Name," "fname," "client_first," "Owner First," etc. as the same field. One row per dog; repeat the owner's info on each row if they have multiple dogs.
