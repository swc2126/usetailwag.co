How to export from Gingr

Show export guide ▾
1. Owners Report
Admin → Reports → Owner Reports → Export CSV
Save as: owners.csv
Columns you'll see: owner_id, first_name, last_name, email, cell_phone, address
2. Animals Report
Admin → Reports → Animal Reports → Export CSV
Save as: animals.csv
Columns you'll see: animal_id, a_first, breed, personality, vaccination_records
3. Reservations Report (Last 12 months)
Admin → Reports → Reservation Reports → set date range to last 12 months → Export CSV
Save as: reservations.csv
Columns you'll see: id, owner_id, animal_id, o_last, a_first, type, check_in_stamp, check_out_stamp, status_string
💡 Tip: The reservations file is what links your animals to their owners. All three files are needed for a complete import.
